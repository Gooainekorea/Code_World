
# 거시적 환경 (Macro Environment)
## 지질환경 (Geological Environment) - OS외 
    코드 월드의 모든 국가는 공중에 떠 있지 않습니다. 운영체제(OS)나 그외의 환경에 맞춰 도시를 건설해야 합니다. 
    하지만 이 지반의 성질(시스템 콜, API, 파일 시스템등)은 지역마다 판이하게 다릅니다.
    때문에 윈도우에서 온 long 주민이 리눅스 지반에서 활동하려다 옷등의 사이즈가 맞지 않아 터져버리는 사고(Portability Issue)가 빈번합니다.(옷이 터집니다. 주민이 터지는건 잔인합니다.)

### 리눅스 광야 [Linux Wilderness](/01_Macro_Environment/Geological_Environment/01_Linux_Wilderrness/01_Linux_Wilderness.md)
    흙을 파면 바로 커널(Kernel)이 드러나는 노천광산입니다. 땅을 파서 구조를 변경할 수 있습니다.

#### 임베디드 개척지 [Embedded Frontier]
    리눅스 광야 안에 존재하는 자원이 극단적으로 적은 오지입니다.

##### 라즈베리숲 (Raspberry Forest)
    임베디드 개척지중 비교적 자원이 풍부해 다른 주민들도 활동가능한 지역입니다.

### 윈도우 고원 (Windows Plateau)
    `NTFS`라는 매우 견고하지만 복잡한 지층으로 이루어져 있습니다. Win32 API 업체의 허가하에 건물 건설이 가능합니다.

### 맥정원 (Mac Darwin Garden)
    유닉스 기반암위에 Cocoa라는 인공 흙으로 덮인 폐쇠적인 인공 정원입니다.
